Polityka Prywatności — Deutsch Lernen

Ostatnia aktualizacja: 28 marca 2026

Aplikacja Deutsch Lernen została stworzona przez Beagle Apps Studio. Niniejsza polityka prywatności wyjaśnia, w jaki sposób aplikacja zbiera, wykorzystuje i chroni dane użytkowników.

1. Jakie dane zbiera aplikacja?

Aplikacja Deutsch Lernen przechowuje dane wyłącznie lokalnie na urządzeniu użytkownika. Są to:

  • Postępy w nauce (punkty XP, poziom, wyniki quizów)
  • Statystyki nauki (liczba wyuczonych słówek, seria dni nauki)
  • Ustawienia aplikacji (motyw, poziom językowy, dzienny cel)
  • Własne słówka dodane przez użytkownika
  • Klucz API Google Gemini (jeśli wprowadzony przez użytkownika)

Aplikacja NIE zbiera danych osobowych takich jak imię, nazwisko, adres e-mail, lokalizacja ani dane demograficzne.

2. Usługi zewnętrzne

Aplikacja korzysta z następujących usług zewnętrznych:

  • Google Gemini API — wykorzystywana w funkcji Czat AI do generowania odpowiedzi nauczyciela AI. Treści rozmów są wysyłane do serwerów Google w celu uzyskania odpowiedzi. Korzystanie z tej funkcji wymaga podania własnego klucza API przez użytkownika. Polityka prywatności Google: policies.google.com/privacy
  • Web Speech API — wykorzystywana do syntezy mowy (wymowa słówek) i rozpoznawania mowy (ćwiczenie wymowy). Dane głosowe mogą być przetwarzane przez usługi przeglądarki.

3. Przechowywanie danych

Wszystkie dane użytkownika są przechowywane lokalnie na urządzeniu za pomocą mechanizmu localStorage przeglądarki. Dane nie są przesyłane na żadne serwery Beagle Apps Studio. Użytkownik może w każdej chwili usunąć swoje dane poprzez funkcję „Resetuj wszystko" w ustawieniach aplikacji.

4. Udostępnianie danych

Beagle Apps Studio nie sprzedaje, nie udostępnia ani nie przekazuje danych użytkowników żadnym podmiotom trzecim, z wyjątkiem sytuacji opisanych w punkcie 2 (komunikacja z Google Gemini API w ramach funkcji czatu).

5. Bezpieczeństwo

Klucz API Google Gemini jest przechowywany lokalnie na urządzeniu użytkownika. Zalecamy nieudostępnianie klucza API innym osobom. Aplikacja nie przechowuje klucza na żadnych zewnętrznych serwerach.

6. Dzieci

Aplikacja nie jest skierowana do dzieci poniżej 13 roku życia. Nie zbieramy świadomie danych od dzieci poniżej tego wieku.
